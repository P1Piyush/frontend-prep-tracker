# 🚀 Frontend Interview Prep Tracker

A premium **60-day structured curriculum** to systematically prepare for frontend engineering interviews. Built with a sleek Zen "Ultra Instinct" design, this tracker helps you stay organised, measure progress, and stay consistent through your entire prep journey.

---

## ✨ Features

- 📅 **60-Day Structured Curriculum** — Day-by-day topics covering everything from JavaScript fundamentals to system design
- ✅ **Progress Tracking** — Mark topics as completed and visualise your advancement at a glance
- 🎨 **Zen Ultra Instinct UI** — Clean, minimal, distraction-free interface designed for focus
- 🔄 **Tracker Update Script** — Python utility to programmatically update and manage your tracker data
- 💾 **Offline-First** — Pure HTML file, no server required; open it in any browser and get started

## 🚀 Getting Started

### Option 1 — Just open the HTML (recommended)

No installation needed. Simply open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/P1Piyush/frontend-prep-tracker.git
cd frontend-prep-tracker

# Open in your default browser
open frontend-prep-tracker.html          # macOS
xdg-open frontend-prep-tracker.html     # Linux
start frontend-prep-tracker.html         # Windows
```

### Option 2 — Use the Python update script

If you want to modify or sync tracker data programmatically:

```bash
# Requires Python 3
python3 tracker_update.py
```

## 📚 60-Day Curriculum Overview

The tracker covers a comprehensive set of topics spread across 60 days:

| Phase | Days | Topics |
|-------|------|--------|
| **Foundation** | 1–10 | HTML5, CSS3, Box Model, Flexbox, Grid |
| **JavaScript Core** | 11–25 | ES6+, Closures, Prototypes, Async/Await, Event Loop |
| **React & Ecosystem** | 26–38 | React fundamentals, Hooks, Context, Redux, React Query |
| **Performance** | 39–45 | Web Vitals, Lazy loading, Code splitting, Caching |
| **System Design** | 46–52 | Component design, API design, Scalability patterns |
| **DSA for Frontend** | 53–58 | Arrays, Trees, Graphs (JS-focused) |
| **Mock Interviews** | 59–60 | Full mock sessions & review |
